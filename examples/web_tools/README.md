# Web Tools (Common Tools)

This example demonstrates the opt-in `penguiflow.common_tools.web` pack.

## Run

```bash
uv run python examples/web_tools/flow.py
```

## Optional: Brave API key

To enable Brave-powered search/context/answers, set one of:
- `BRAVE_SEARCH_API_KEY` (preferred)
- `BRAVE_API_KEY`
